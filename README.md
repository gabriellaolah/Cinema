Project Name: Husets Biograf Website (Retro Cinema Site)
A custom website concept for Husets Biograf, a quirky underground cinema in Copenhagen that screens cult classics, 80s/90s films, and VHS tapes. The site is designed to reflect the cinema’s unique, nostalgic atmosphere — blending vintage aesthetics with dynamic content and playful UI.

Project Goals
Build a fully dynamic site that reflects the personality of the cinema
Use a retro-inspired design influenced by VHS, glitch effects, and analog visuals
Make it easy for non-technical staff to update the programme and media content

Tech Stack
Frontend: HTML, CSS (Tailwind), JavaScript (Vite)
CMS: Strapi (headless CMS for managing film programme)
Image Handling: Working with media content stored in Strapi
Planned Integrations: Billetto for ticket links

Features (in progress)
“Now Showing” section fetching dynamic data from Strapi
“The Vault” archive for past events and movies
VHS glitch-style animations and a custom 404 page
Retro layout with a modern responsive structure (in progress)
